Aplikasi Helpdesk sederhana berbasis web menggunakan **Streamlit**, **TiDB Cloud (MySQL)**, dan dikelola via **GitHub**.

## Fitur
- Input tiket keluhan user.
- Dashboard monitoring status tiket.
- Database real-time dengan TiDB Cloud.

## Cara Menjalankan Lokal
1. Clone repo ini.
2. `pip install -r requirements.txt`.
3. Isi `.streamlit/secrets.toml` dengan kredensial database kamu.
4. Jalankan: `streamlit run app.py`.# it-helpdesk-app
